## 🕹️ Agentic AI, RAG & Semantic Patterns

- 🤝 Design **agentic AI workflows** where Joule and custom SAP AI agents plan, call tools/APIs, and execute workflows autonomously across SAP and non‑SAP systems.
- 🔁 Use **plan–act–reflect** loops with budget caps, deterministic fallbacks, and durable state instead of “one giant magic prompt”.
- 📚 Build **RAG + semantic search** stacks with hybrid retrieval (BM25 + vector search), semantic chunking, reranking, and freshness windows tuned to the corpus, not the hype cycle. 

## 🧱 Patterns I Care About

- 🧩 **RAG architecture**  
  Hybrid search, query rewriting, semantic chunking, reranking, and freshness windows — picked for the corpus, not for fashion. `pgvector • BM25 • rerankers • eval sets`

- 📡 **Protocols (MCP design)**  
  MCP‑style server federation, thoughtful tool‑surface design, prompt‑as‑resource patterns, and scoped auth with progressive disclosure. `MCP • tools • OAuth scopes`

- 🧠 **Agents (agentic workflows)**  
  Plan‑act‑reflect loops with budget caps, deterministic fallbacks, and durable planner state — one workflow at a time, not a single prompt. `Tool use • PlannerState`

- ✅ **Quality (eval harnesses)**  
  Offline regression sets, golden traces, LLM‑as‑judge calibrated against humans, and prod canaries wired into every shipped change. `Braintrust • Promptfoo • tracing`

- 🎛️ **Models (selection & routing)**  
  Match **Claude**, **Joule**, and fine‑tuned open models to the task class; pick cost/latency/quality per call, not per project. `Claude • Joule • OSS LLMs`

- 🏢 **Enterprise (SAP Joule extensibility)**  
  Joule extensions, grounding on SAP business data (BTP, Datasphere), governance‑first prompts, and clean handoffs into back‑office systems. `Joule • BTP • Datasphere`

- 🎯 **Tuning (fine‑tuning & adapters)**  
  When prompts plateau: SFT, LoRA, DPO, distillation — always with eval deltas, never on vibes. `LoRA • DPO • SFT`

- 🛡️ **Guardrails (safety & risk)**  
  Policy‑tuned classifiers, structured‑output validation, PII redaction, and refusal calibration mapped to the risk register. `Classifiers • Schemas • PII filters`

- 🧰 **Platform (AI platform engineering)**  
  Prompt registries, eval pipelines, observability, secret rotation, and all the boring plumbing that makes pilots productionizable. `Tracing • Registry • CI/CD`

- 🧠 **Context (long‑context patterns)**  
  Prompt caching, structured summaries, retrieval‑as‑window, and discipline about what actually deserves a token. `Caching • summary maps`

- 🧭 **Discovery (AI roadmap & opportunity sizing)**  
  Workshops to find the 3–5 use cases that justify a quarter of investment, with success metrics agreed and measured up front.
