## 🧱 Patterns I Care About

- 🧩 **RAG architecture**  
  Hybrid search, query rewriting, semantic chunking, reranking, and freshness windows — picked for the corpus, not for fashion. `pgvector • BM25 • rerankers • eval sets`[web:40][web:43][web:52]

- 📡 **Protocols (MCP design)**  
  MCP‑style server federation, thoughtful tool‑surface design, prompt‑as‑resource patterns, and scoped auth with progressive disclosure. `MCP • tools • OAuth scopes`[web:27][web:35]

- 🧠 **Agents (agentic workflows)**  
  Plan‑act‑reflect loops with budget caps, deterministic fallbacks, and durable planner state — one workflow at a time, not a single prompt. `Tool use • PlannerState`[web:41][web:44][web:50][web:47]

- ✅ **Quality (eval harnesses)**  
  Offline regression sets, golden traces, LLM‑as‑judge calibrated against humans, and prod canaries wired into every shipped change. `Braintrust • Promptfoo • tracing`[web:42][web:45][web:48][web:51][web:54]

- 🎛️ **Models (selection & routing)**  
  Match **Claude**, **Joule**, and fine‑tuned open models to the task class; pick cost/latency/quality per call, not per project. `Claude • Joule • OSS LLMs`[web:24][web:18][web:25]

- 🏢 **Enterprise (SAP Joule extensibility)**  
  Joule extensions, grounding on SAP business data (BTP, Datasphere), governance‑first prompts, and clean handoffs into back‑office systems. `Joule • BTP • Datasphere`[web:24][web:19][web:37]

- 🎯 **Tuning (fine‑tuning & adapters)**  
  When prompts plateau: SFT, LoRA, DPO, distillation — always with eval deltas, never on vibes. `LoRA • DPO • SFT`[web:48][web:54]

- 🛡️ **Guardrails (safety & risk)**  
  Policy‑tuned classifiers, structured‑output validation, PII redaction, and refusal calibration mapped to the risk register. `Classifiers • Schemas • PII filters`[web:51][web:54]

- 🧰 **Platform (AI platform engineering)**  
  Prompt registries, eval pipelines, observability, secret rotation, and all the boring plumbing that makes pilots productionizable. `Tracing • Registry • CI/CD`[web:51][web:54]

- 🧠 **Context (long‑context patterns)**  
  Prompt caching, structured summaries, retrieval‑as‑window, and discipline about what actually deserves a token. `Caching • summary maps`[web:39][web:52]

- 🎙️ **Voice (multimodal & voice)**  
  Real‑time transcription, speech‑to‑action mapping, document understanding, and screen‑reading agents for back‑office workflows. `STT • vision • OCR`[web:31][web:36]

- 🧭 **Discovery (AI roadmap & opportunity sizing)**  
  Workshops to find the 3–5 use cases that justify a quarter of investment, with success metrics agreed and measured up front.[web:41][web:51]
